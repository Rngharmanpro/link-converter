# link-converter
A web service that helps others to convert links between mobile and web applications.

## Requirements

- Java 1.8
- Zookeeper & Kafka
- Mysql

## To run project

- ./gradlew build
- java -jar build/libs/link-converter-0.0.1-SNAPSHOT.jar

## To run all tests
- ./gradlew test
